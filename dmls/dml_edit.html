This function purpose is to edit an existing row of data

  • "UPDATE companies SET Age=15;"


    This code implies that you try to edit all existing data of age in companies and set it into 15 with no excuses.

  • "UPDATE companies SET Age=15 WHERE Name='Andy';"


    This code implies that you try to edit a row of data with the name 'Andy'. Using where will help a lot because you won't edit the entire table.
